# 路线图 (ROADMAP)

> 长期规划，持续迭代

## v1.0 MVP
**目标**：核心刷题体验，快速上线

### 功能
- [x] 五大行测模块切换
- [x] 随机出题（每模块至少 20 题）
- [x] 单题倒计时（按模块设定）
- [x] 选完立即判分（显示正确答案）
- [x] 随时退出，进度自动保存到 localStorage
- [x] 基础统计（本次正确率）

### 技术
- 单页 HTML，部署到 GitHub Pages
- 题库存储在 `questions/*.json`
- 用户数据存 localStorage（未来迁移到 Gist）

## v1.3 知识点库
**预计时间**：v1.2 后 2 周

### 功能
- [x] 知识点按模块浏览
- [x] 关键词搜索知识点
- [x] 做题错后关联相关知识点
- [x] 用户投喂知识点（Mavis 整理入库）

### 知识点格式
```markdown
## 模块：常识判断
### 知识点：民事责任年龄

**三个阶段：**
1. 无民事行为能力人：< 8岁
2. 限制民事行为能力人：8 ~ 18岁
3. 完全民事行为能力人：>= 18岁

**相关真题**：题号 #23, #45
```
