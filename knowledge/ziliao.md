# 资料分析知识点

> 来源：粉笔 2027年山东国省考基础精讲班
> 更新时间：2026-06-25

## 一、增长

### 1. 增长量
```
增长量 = 现期值 - 基期值
       = 基期值 × 增长率
       = 现期值 × 增长率 / (1 + 增长率)
```

### 2. 增长率
```
增长率 = (现期值 - 基期值) / 基期值
       = 增长量 / 基期值
       = 增长量 / (现期值 - 增长量)
       = 现期值 / 基期值 - 1
```

### 3. 基期值
```
基期值 = 现期值 - 增长量
       = 现期值 / (1 + 增长率)
       = 增长量 / 增长率
```

### 4. 现期值
```
现期值 = 基期值 + 增长量
       = 基期值 × (1 + 增长率)
       = 增长量 / 增长率 × (1 + 增长率)
```

## 二、倍数

### 1. 基本倍数
```
A是B的几倍 = A / B
A比B多几倍 = A / B - 1
```

### 2. 基期倍数
```
基期A是基期B的几倍 = (A - A的增长量) / (B - B的增长量)
                   = A/B × (1 + B的增长率) / (1 + A的增长率)
```

### 3. 翻番
```
A翻n番 = A × 2^n
```

## 三、比重

### 1. 基本公式
```
比重 = 部分值 / 整体值
部分值 = 整体值 × 比重
整体值 = 部分值 / 比重
```

### 2. 基期比重
```
基期比重 = (部分值 - 部分值增长量) / (整体值 - 整体值增长量)
         = 部分值/整体值 × (1 + 整体值增长率) / (1 + 部分值增长率)
```

### 3. 比重增长量（变化）
```
比重增长量 = 现期比重 - 基期比重
           = 部分值/整体值 × (部分值增长率 - 整体值增长率) / (1 + 部分值增长率)
           （结果读作"百分点"）
```

### 4. 判断比重变化
| 条件 | 结论 |
|------|------|
| 部分增长率 > 整体增长率 | 比重上升 |
| 部分增长率 < 整体增长率 | 比重下降 |
| 部分增长率 = 整体增长率 | 比重不变 |

## 四、平均数

### 1. 基本公式
```
平均数 = 总量 / 份数
总量 = 平均数 × 份数
份数 = 总量 / 平均数
```

### 2. 基期平均数
```
基期平均数 = (总量 - 总量增长量) / (份数 - 份数增长量)
           = 总量/份数 × (1 + 份数增长率) / (1 + 总量增长率)
```

### 3. 平均数增长量
```
平均数增长量 = 现期平均数 - 基期平均数
             = 总量/份数 × (总量增长率 - 份数增长率) / (1 + 总量增长率)
```

### 4. 平均数增长率
```
平均数增长率 = (总量增长率 - 份数增长率) / (1 + 份数增长率)
```

### 5. 判断平均数变化
| 条件 | 结论 |
|------|------|
| 总量增长率 > 份数增长率 | 平均数增加 |
| 总量增长率 < 份数增长率 | 平均数减少 |
| 总量增长率 = 份数增长率 | 平均数不变 |

## 五、隔年增长

### 1. 隔年增长率
```
已知：相邻两年增长率分别为 q₁ 和 q₂
隔年增长率 = q₁ + q₂ + q₁ × q₂
```

### 2. 隔年倍数
```
隔年倍数 = 1 + q₁ + q₂ + q₁ × q₂
```

### 3. 隔年基期值
```
隔年增长中，基期值 = 现期值 / (1 + q₁ + q₂ + q₁ × q₂)
```

## 六、年均增长

### 1. 年均增长量
```
年均增长量 a = (末期值 - 初期值) / n
               （n为增长次数/年份差）
```

### 2. 年均增长率
```
年均增长率 q = n√(末期值/初期值) - 1
```

### 3. 相关计算
```
末期值 = 初期值 × (1 + q)^n
初期值 = 末期值 / (1 + q)^n
       = 末期值 - a × n
```

## 七、常见统计指标

### 1. GDP相关
```
国内生产总值(GDP) = 第一产业增加值 + 第二产业增加值 + 第三产业增加值
```

### 2. 人口相关
```
人口自然增长率 = (年内出生人口 - 年内死亡人口) / 年平均人口 × 1000‰
              = 人口出生率 - 人口死亡率
```

### 3. 进出口相关
```
进出口总额 = 出口额 + 进口额
贸易顺差 = 出口额 - 进口额（出口 > 进口）
贸易逆差 = 进口额 - 出口额（进口 > 出口）
```

### 4. 其他指标
```
贡献率 = 部分增长量 / 整体增长量 × 100%
拉动增长 = 部分增长量 / 整体基期值 × 100%（结果读作百分点）
```

## 八、常见比率

| 比率 | 公式 |
|------|------|
| 产销率 | 销售数量 / 生产数量 × 100% |
| 工业产销率 | 工业销售产值 / 工业总产值 × 100% |
| 利润率 | 利润 / 收入 × 100% |
| 资产负债率 | 负债合计 / 资产总计 × 100% |
| 上座率 | 上座人数 / 总座位数 × 100% |

## 九、速算技巧

### 1. 增长率相关
- 若增长率很小（<5%），可用"增长量 ≈ 基期值 × 增长率"
- 分数比较：分子分母同时变化时，看"差分法"

### 2. 比重相关
- 两期比重差 = A/B × (a-b)/(1+a)，结果读作百分点
- 判断升降：比较 a 和 b 的大小即可

### 3. 平均数相关
- 两期平均数差 = A/B × (a-b)/(1+a)
- 平均数增长率 = (a-b)/(1+b)
